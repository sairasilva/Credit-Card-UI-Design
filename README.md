# 💳 Credit Card UI Design

## ✨ Descrição do Projeto
Este projeto apresenta uma interface moderna de cartão de crédito desenvolvida com HTML e CSS.  
O layout simula um cartão real, exibindo informações como número, nome do titular e data de validade.  
O design é limpo, elegante e utiliza imagem de fundo para criar um visual mais realista.  

---

## 🎯 Objetivo
- Praticar a estruturação com HTML5  
- Desenvolver estilização com CSS3  
- Criar interfaces visuais modernas e organizadas  
- Trabalhar com imagens de fundo e tipografia  
- Simular elementos reais de forma visual  

---

## 🧠 Tecnologias Utilizadas
- HTML5  
- CSS3  
- Google Fonts (Poppins)  

---

## 🚀 Funcionalidades
- Interface de cartão de crédito estilizado  
- Exibição de número, nome e validade do cartão  
- Uso de imagem de fundo personalizada  
- Layout centralizado na tela  
- Design moderno e minimalista  
- Tipografia personalizada com Google Fonts
