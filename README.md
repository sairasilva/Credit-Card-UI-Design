💳 Credit Card UI Design

#

✨ Descrição do Projeto

Este projeto apresenta uma interface moderna de cartão de crédito desenvolvida com HTML e CSS. O layout simula um cartão real, exibindo informações como número, nome do titular e data de validade, com um design elegante e minimalista.

Diferente da versão com animação, este modelo foca em um visual estático, com uso de imagem de fundo e estilização refinada.

#

🎯 Objetivo

O objetivo deste projeto é:

Praticar estruturação com HTML5
Explorar estilização com CSS3
Criar interfaces visuais realistas
Trabalhar com imagens de fundo e tipografia
Desenvolver layouts organizados e modernos

#

🧠 Tecnologias Utilizadas

HTML5
CSS3
Google Fonts (Poppins)

#

🚀 Funcionalidades

💳 Interface de cartão de crédito estilizado
🎨 Uso de imagem de fundo personalizada
🖌️ Design limpo e moderno
📱 Layout centralizado e adaptável
✍️ Tipografia elegante com Google Fonts
